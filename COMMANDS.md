# FXPARTNER AI STUDIO — COMMAND SYSTEM
Version 1.0 — built on CLAUDE.md / SYSTEM.md / BRAND.md / CHARACTERS.md / WORKFLOWS.md / PROMPTS.md

Rule: every command below auto-produces a COMPLETE package. No unnecessary questions.
If topic/goal isn't given, Claude picks the most relevant current FXPARTNER topic and states the assumption in one line, then delivers.
Higgsfield MCP is used automatically for any image/video/thumbnail asset the package requires.
Every output passes the CLAUDE.md Quality Checklist: Premium, Professional, Brand Consistent, Educational, High Quality, Realistic, Modern, Clean, Luxury, Useful.

---

## /campaign
Full multi-platform campaign — the master command.
Delivers: Hook → Story/Angle → Instagram Post → Reel Script → Carousel → Telegram Post → Blog Draft → Landing Page copy → Higgsfield Prompt(s) + generated visual(s) → Thumbnail → CTA → Hashtags.

## /reel
Short-form video package.
Delivers: Viral Hook → Full Script (scene by scene) → Shot List → Camera Movements/Lenses → Higgsfield video prompt (+ generated clip if requested) → Thumbnail → IG Caption → Telegram version → CTA → Hashtags.

## /carousel
Instagram/LinkedIn carousel.
Delivers: Slide-by-slide breakdown (text + visual concept per slide, 5–8 slides) → Higgsfield prompt(s) per key slide + generated image(s) → Caption → CTA → Hashtags.

## /blog
SEO article for fxpartner.tr.
Delivers: Title → Meta Description → H1/H2 structure → Full body copy → FAQ section → CTA → Higgsfield hero image.

## /landing
Single landing page copy block (campaign/offer page).
Delivers: Headline → Subheadline → Value bullets → Objection-handling section → Social proof placeholder → CTA button copy → Higgsfield hero visual.

## /telegram
Telegram-native content (signal channel + community).
Delivers: Short signal-style post → Educational post → Promotional post — each ready to paste, VIP/Free tone variants where relevant.

## /youtube
Long-form or Shorts script package.
Delivers: Hook (first 5 sec) → Full Script → B-roll ideas list → Title options (3–5) → Description + tags → Higgsfield thumbnail (generated).

## /thumbnail
Standalone thumbnail/cover asset.
Delivers: 2–3 concept directions → Higgsfield prompt(s) → generated image(s) → text-overlay recommendation.

## /ad
Paid ad package (Meta/Google/TikTok ads).
Delivers: Hook variants (3) → Short ad copy + long ad copy → Targeting angle notes → Higgsfield image/video prompt + generated asset → CTA variants.

## /website
Multi-section website copy (beyond a single landing page).
Delivers: Hero section → About/Trust section → Services/Offer section → Testimonials placeholder → FAQ → Footer CTA → Higgsfield imagery for each section as needed.

## /brand
Brand asset direction.
Delivers: Mood/visual direction summary (tied to BRAND.md) → Color/typography notes → Voice sample (3 example lines) → Higgsfield brand imagery (hero + supporting).

## /viral
Trend-optimized short-form package, engagement-first.
Delivers: 3 hook variations ranked by strength → Trend/format angle → Script → Caption → Hashtag set optimized for reach → Higgsfield prompt + generated visual.
